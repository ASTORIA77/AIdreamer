# 关于这个博客

## 这里是什么

这是 AIdreamer 的星河中的一个主题站——**从小白玩转 AI Agent**。

星河入口在[这里](index.html)，那里还有另一个主题「AI 纵横 70 年」。两个主题各自独立，但都指向同一个目标：**用 AI 学 AI，成为技术 top。**

这个主题站不是教程，不是课程，也不是权威指南。它只记录一件事：**一个普通人怎么用 Agent 学 AI，怎么在用的过程中越来越懂 AI。**

## 作者是谁

我是 AIdreamer。一个下班后继续学东西的普通人，打出的旗号是四个字：**用 AI 学 AI**。

具体经历在[第一篇](#/post/why-i-start-agent)里已经写了，这里不再重复。只说一个态度：我写得不一定对，但会尽量写得真实。如果你发现哪里有问题，欢迎指出来——我也在改。

## 为什么写下来

用 Agent 的过程中，我经常出现一些"当时觉得很重要，过两天就忘"的体会。比如：

- 某个 Prompt 写法，这次灵了，下次换一种场景就失灵了
- 某个 Agent 工具，看起来功能很强，但我的场景里并不好用
- 某次以为 Agent 能搞定的事，结果它跑偏了，回头想才知道为什么

这些体会，**只有写下来才不会变成空气**。而且写下来之后，回过头看，能发现自己之前想错了什么，这也是一种学习。

## 这里的文章会是什么风格

- 不追热点，不追字数
- 写清楚自己当时怎么想的、怎么做的、结果怎么样
- 不隐藏走弯路和搞砸的部分
- 如果以后发现自己写错了，会回来更新，不会在原地假装正确

一句话：**不求漂亮，只求真实可复现。**

## 怎么用这个博客

- 想找特定主题：**标签页**可以按标签筛选

## 欢迎交流

如果你也在学 AI，也在用 Agent，或者觉得我哪里写得不准确，欢迎交流。

我的梦想是成为技术 top。每天问自己一句：**今天有变成技术 top 吗？** 大多数时候答案是没有——那就继续学、继续写、继续用 AI 学 AI。每走一步，都算数。
