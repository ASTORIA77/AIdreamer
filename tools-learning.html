LEARNING

AI 学习

线上资源多到学不完，问题不在于找不找得到

这个时代学 AI，缺的从来不是资源——缺的是你想学的那股劲。课程、平台、社区、算力，全是免费的或者白菜价。只要你想学，就学得了。

课程

吴恩达 · deeplearning.ai

学过

吴恩达的课是很多人 AI 入门的起点，我就是看他的课起步的。2026 年 5 月他又出了新课——《AI Prompting for Everyone》，3 小时，零代码，面向所有人。在 deeplearning.ai 平台上，有付费有免费，课程质量在全球范围内都是第一梯队。

B 站上搜「吴恩达」「机器学习」有一大堆搬运和汉化版本，涵盖监督学习、非监督学习、神经网络基础全套。如果你英语还行，直接去 deeplearning.ai 官网看原版，体验更好。

廖雪峰 · Python

正在学

学 AI 绕不开 Python。我现在正在看廖雪峰的 Python 教程——中文教程里逻辑最清晰的一个，从基础语法到常用库都有覆盖，实战导向。网上免费的版本到处是，跟着敲一遍基本够用了。

B 站 · 中国大学 MOOC

用过

B 站是个巨大的 AI 教室——搜「人工智能」「深度学习」「LLM」，国内外的课几乎都能找到搬运或者中文精讲。国内的大学课去中国大学 MOOC（icourse163.org），清华北大上交的经典课程基本都上线了。国外的斯坦福 CS229（机器学习）、CS231n（计算机视觉）也能在 YouTube 和各大公开课平台找到完整版本。

资源大把大把的，关键是你有没有坐下来打开第一个视频的冲动。

上手实践

PaddlePaddle · 云平台

了解中

光看课不写代码等于没学。百度飞桨（PaddlePaddle）有免费的 GPU 算力资源给学习者和开发者用，不用自己买显卡也能跑模型训练。各大云平台——阿里云、腾讯云、华为云——也都有学生免费额度或者极低价格的学习套餐。

写一整页代码、报一整页错、改一整页 Bug——这个过程比看十节课都管用。算力不是门槛，愿不愿意动手才是。

GitHub · Git · Gitee

在用

学 AI 绕不开版本控制。找开源项目、复现论文代码、管理自己的学习记录——全都需要 Git。

GitHub 是生态最好的，全球 AI 开源项目的首选托管平台。国内访问经常不稳定，但你不是第一个遇到这个问题的人——直接问 DeepSeek「怎么上 GitHub」，它会给出一堆靠谱的办法。国内替代是码云（Gitee），我用过，中文界面友好，国内访问快，不过上面大部分是镜像仓库，原创开源项目体量和质量跟 GitHub 还是有差距。

Git 本身不难学——clone、commit、push、pull，四个命令够你用了。

CSDN · 各大技术社区

日常在看

遇到报错怎么办？把错误信息复制到搜索引擎，大概率 CSDN、知乎、博客园、SegmentFault 上已经有人踩过同样的坑了。AI 相关的讨论在 GitHub Issues、Reddit r/MachineLearning、Hugging Face 论坛也都很活跃。

2026 年学 AI，学习路径、工具、社区这三样全都拉满了——你想走的每一步，几乎都有人帮你趟过路了。剩下的，只是你愿不愿意迈出第一步。
