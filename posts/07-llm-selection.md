## 一、缘起：七十年长跑的最后一公里

大语言模型并非横空出世。它是一场跨越 70 年的接力赛，由统计学家、语言学家、计算机科学家共同跑出，最后在 2017 年的一篇 8 页论文中迎来转折。

| 年代 | 事件 |
|------|------|
| 1950-1980 | 符号主义与连接主义的拉锯。图灵测试问世，感知机被证伪，专家系统兴起又陷入冬天 |
| 1986-2012 | 反向传播复活神经网络，深度学习破晓。Hinton 等人重启 BP 算法；2012 年 AlexNet 在 ImageNet 一战成名 |
| 2017 | 《Attention is All You Need》发表，Transformer 架构发布——LLM 真正的起点 |
| 2018-2020 | BERT 与 GPT 分流。理解派 vs 生成派；GPT-3 以 1750 亿参数验证「Scaling Law」可行 |
| 2022.11 | ChatGPT 出圈。RLHF 让模型「会说话」，五日破百万用户 |
| 2023-2025 | 百模大战与开源觉醒。Llama、Qwen、DeepSeek 开源生态崛起 |
| 2026 | 国产闭源向开源「破壁」。DeepSeek V4 万亿 MoE 落地，与华为昇腾深度耦合 |

**什么是「生成式 AI」**？判别式 AI 回答「这是什么」；生成式 AI 回答「下一个会是什么」。前者擅长分类与识别，后者擅长创造与续写。LLM 正是生成式范式在文本上的极致体现——它不是在「检索」，而是在按概率「编织」。

> 模型不存储知识，模型存储概率。它的每一句话，都是一次基于上下文的最大可能选择。

## 二、群像：按生态位读懂中国大模型版图

国内大模型不是一片混战，而是生态分层。按「身位」分为三类：**云厂派、纯模派、跨界派**。读懂生态位，选型就成功了一半。

### 云厂派 · 算力与场景的合体

背靠云基础设施与海量业务场景，模型与平台深度绑定，胜在「全栈」与「企业级」。

**通义千问 Qwen（阿里云）**：开源生态最完整的国产模型，Qwen3 系列覆盖 0.5B 到千亿，HuggingFace 下载榜常年前列。优势在开源开放、多语种强、全模态布局；阿里云百炼平台一站式接入。

**豆包 Doubao（字节跳动）**：背靠抖音/今日头条的内容生态，C 端产品打磨细腻，价格屠夫起家。中文创作、视频脚本、短剧生成是天赋型选手；火山引擎接入便利。**2026 年 6 月 23 日发布 Seed 2.1 系列（Pro 和 Turbo），官方称 Pro 在评测数据上已稳定超过 Opus 4.6，部分场景与 4.7/4.8 持平，"跨越了生产力质变临界点"。这个"临界点"说法我没测过，不好下结论，但豆包的迭代速度确实凶猛。**

**混元 Hunyuan（腾讯）**：沉稳务实路线，深度嵌入腾讯文档、企业微信、QQ 浏览器等场景。长文档处理、办公场景集成是强项；近期开源 Hunyuan-Large MoE。**腾讯元宝近期没啥大动向。**

**文心一言 ERNIE（百度）**：国内最早 All in LLM 的一线大厂，知识图谱底子厚，搜索与问答场景强势。行业模型积累深、政企落地经验丰富；近期开源 ERNIE 4.5 系列。

### 纯模派 · 「这一家很特殊」

**DeepSeek 深度求索**——无云、无 C 端、无大厂背景，把所有筹码压在算法本身。一家公司带火了「MoE+稀疏注意力」路线，硬是把训练成本打下了一个数量级。

2026 年 4 月发布的 **DeepSeek-V4** 万亿参数 MoE，引入 DSA（DeepSeek Sparse Attention）稀疏注意力，长上下文能力跃升。已成内部 Agentic Coding 主力，体验逼近 Claude Opus 4.6 非思考模式，并明确携手华为昇腾，为国产算力时代铺路。

**关于 API 价格**：DeepSeek V4 正式版定档 2026 年 7 月中旬上线，将引入**峰谷定价机制**——高峰时段（工作日白天部分时段）API 价格翻倍，非高峰时段维持原价。这也是大模型行业首个峰谷定价，用价格杠杆调节算力挤兑。对个人用户来说，**非高峰期（晚上和周末）调 API 更便宜**。

> DeepSeek 代表「中国式工程美学」：少而精、深而专、开源开放、技术报告诚意十足。

### 纯模派 · 各擅胜场的四星

**智谱 GLM（清华系）**：学术血统纯正，**目前已更新到 GLM-5.2**，在中文理解、Agent 能力上稳居第一梯队，To B 政企落地踏实。官方网页 chat.z.ai 体验干净，海外品牌 Z.ai 表现亮眼。

**Kimi（月之暗面）**：以「超长上下文」立身，**近期刚发布了 K2.7 Code**，开源、1T 参数，专攻编程场景，在代码 Benchmark 上常爆出彩成绩。网页端简洁优雅，文件读取与长文档摘要是杀手锏。

**MiniMax（多模态见长）**：**2026 年 6 月 1 日正式发布并开源 M3 模型**，428B 参数，原生多模态，1M 超长上下文，主打前沿 Coding 能力。语音合成 + 视频生成走在国内前列，海螺 AI 是其 C 端门面。

**阶跃星辰 Step（端侧多模态）**：主推 Step3-VL-10B（10B 参数开源多模态）和 Step 3.7 Flash（生产级 Agent Flash 模型），走的是「把强模型做小、塞进端侧」的路线。与吉利汽车深度绑定，车机端口落地强。

### 跨界派 · 业务驱动的「副业大模型」

**LongCat（美团）**：主打 Agent、本地生活与外卖调度场景，开源版本已对外发布，工程务实。

**MiMo（小米）**：从手机端 + 车机 + IoT 三端协同切入，早期确实是端侧小模型路线。但 2026.4 发布的 MiMo-V2.5 已经是万亿参数 MoE（总参 1.02 万亿、激活 420 亿），旗舰 Pro-UltraSpeed 版还刷新了全球最快推理纪录。现在是「云端旗舰 + 端侧轻量」双线并行，不再只是"小而灵"了。

**京言 JoyAI（京东）**：专注电商 / 客服 / 供应链场景。京东的言犀大模型品牌在 2025 年 WAIC 上正式升级为 JoyAI，内置在京东 App 里的 AI 导购助手叫「京言」，覆盖专业品类咨询、产品对比、购物建议等服务。

> 云厂派比生态，纯模派比算法，跨界派比场景。选型，先看你站在哪座山下。

## 三、渡海：三座灯塔的不同信仰

把目光投向太平洋的彼岸。OpenAI 信「规模」，Anthropic 信「安全」，Google 信「生态」。三家最强的玩家，恰好代表了大模型时代三种最根本的世界观。

### OpenAI · 规模即真理

**GPT-5.5** 是当前公开可用的最新版本（2026.04.23 发布），定位「真实工作的新型智能」，自 GPT-4.5 以来首个从零重训的基础模型。重点不在"更聪明"，而在"能不能把复杂任务持续推进到底"。

**GPT-5.6 系列已公布但尚未全面开放**（2026.06.27）：旗舰版 **Sol**、均衡版 **Terra**、轻量版 **Luna** 三档。其中 Sol 在 Terminal-Bench 2.1 编程基准上得分 88.8%，开启 Ultra 模式后达 91.9%，超过了 Claude Mythos 5。但受美国政府要求，目前仅向少数「可信合作伙伴」开放预览，**GPT-5.6 全面开放还需等待**。

气质：商业化最激进、多模态最齐全、生态最庞大；学术写作、文献综述、跨语言翻译、知识汇总场景的"国际通用语"。

### Anthropic · 安全是头等大事

**Claude Opus 4.8** 是当前 Opus 线最新版本，在 SWE-bench 编程基准上稳居前列。社区共识是「编程问题第一选择」——尤其复杂重构、架构设计、长会话编辑场景。

**Claude Sonnet 5 刚发布**，性能直逼 Opus 4.8，成本更低。被描述为「迄今为止最具自主智能体特性的 Sonnet 版本」，在制定计划、使用浏览器和终端等工具方面显著进步。

**关于「宙斯」Mythos**：课件里原来写的「强到不敢发的模型」现在已经公开了。2026 年 6 月 9 日，Anthropic 发布了 **Claude Fable 5**——这是首个面向公众开放的 **Mythos 级别模型**。此前 Mythos Preview 因能独立发现 IT 系统零日漏洞而"不敢发布"，现在 Fable 5 把这个能力以可控方式开放给了开发者和订阅用户。**最强公开模型不再是 GPT 或 Opus，而是 Fable 5。**

气质：Constitutional AI 思想下的"长远主义者"，长上下文稳、长任务不漂、写代码"思考型"。

### Google · 生态即护城河

Google 是三家中唯一「芯片—框架—模型—应用」全栈自研的玩家，自研 TPU 已到第八代。

模型层面，**Gemini 3.5 Flash** 是当前主力（2026.05 Google I/O 发布），不再是课件里写的 3.1 Pro。最反常识的地方在于：它是"Flash"轻量级定位，性能却反超了 Pro。Gemini 3.5 家族包含 Omni、Flash、Spark 三个版本。

但**近期的表现上，Google 在三家中已经落了下风**——产品迭代节奏不如 OpenAI 凌厉，编程能力不如 Anthropic，国内访问门槛又高。全栈自研是护城河，也是包袱。

气质：工程深度最强、多模态最纯正（从训练第一天就同步喂入文本/代码/图像/音频/视频）、与 Google 全家桶的"无感融入"无人能敌。但差异化定位需要相应场景才能激活。

### 三家速览

| 维度 | OpenAI · GPT-5.5 | Anthropic · Claude Fable 5 | Google · Gemini 3.5 Flash |
|------|---------|---------|---------|
| 世界观 | 规模即真理 | 安全大于能力 | 生态即护城河 |
| 最强场景 | 学术、写作、通用 Agent | 编程、长任务推理、复杂重构 | 多模态（视频/音频）、Workspace 集成 |
| 训练芯片 | NVIDIA（深度合作） | NVIDIA + AWS Trainium | 自研 TPU（全栈自主） |
| 性格隐喻 | 八面玲珑的全才 | 沉稳谨慎的工程师 | 无处不在的图书馆 |
| 国内访问 | 需科学上网 + 付费 | 需科学上网 + 付费 | 需科学上网（部分免费） |

**个人使用心法（一种参考）**：
- 学术研究、文献综述、知识汇总 → 优先 GPT-5.5
- 编程、代码重构、技术方案设计 → 优先 Claude（Opus 4.8 或 Sonnet 5）
- 视频/音频理解、Google 生态内办公 → 用 Gemini

> GPT 像一位博学的演讲者，Claude 像一位沉稳的工程师，Gemini 像一座沉默的图书馆。选谁不是排名问题，是气质匹配问题。

## 四、入口：免费 Chatbot 与那些「藏得很深」的官方网页

绝大多数用户，永远不需要碰 API。但要选对那个「日常打开的网页」，并不是一件简单的事——好的 Chatbot 网页，本身就是一种生产力工具。

### 一个先要承认的事实 · 日常任务，差距已经很小

站在 2026 年回头看，国内第一梯队的模型在普通日常任务上已经高度趋同。让 DeepSeek 写一段产品文案，让智谱清言写同一段，让 Qwen 写同一段——95% 的用户区分不出来。

这意味着两件事：

1. **不必再纠结提示词工程**。曾经那些「魔咒级 Prompt」「请你扮演……」的玄学正在快速退场——好模型不需要你哄，写一句白话它就能干活。
2. **日常场景，挑顺眼的那个就行**。免费、网页快、能联网、说人话，就是好。剩下的差异，多到了"风格"层面，不到"能力"层面。

### 但复杂任务依然会拉开差距 · 看 Benchmark

真正考验模型的，是那些需要长链推理、Agent 调度、多步代码编辑的任务。在这些场景下，差距是真实存在的，而且要靠 benchmark 来辨别：

| 场景 | 该看的基准 | 说明 |
|------|-----------|------|
| 编程能力 | SWE-bench Verified / Pro / Multilingual | Pro 是 Scale AI 出的企业级真实任务版本，难度大幅提升——最强模型在 Verified 上能拿 80%+，到了 Pro 只有 ~46%；OpenAI 已弃用 Verified 转向 Pro |
| Agent 任务 | τ-bench / GAIA / WebArena | 考察多步工具调用、长任务规划、网页操作的稳定性 |
| 推理能力 | AIME 2025 / GPQA Diamond | 奥赛数学与博士级科学问答，是大模型推理深度的"硬指标" |
| 长上下文 | RULER / LongBench v2 | 不是「装得下」就行，要看真的「读得懂、记得住」 |

提示：榜单要看「最近三个月」的版本，旧 benchmark 普遍存在数据污染（模型见过题目）。SWE-bench Verified 被弃用就是典型案例。

### 一个朴素的体感 ·「新模型，新脑子」

抛开所有花哨的方法论，最朴素的一条经验是：**日常任务，永远用最新发布的模型。**

- 新模型的训练数据更新——你问「2026 年 4 月发布的某产品」，老模型只会装懂或承认不知道
- 新模型的对齐更细——同样一个开放性问题，新模型会主动澄清、追问、给出结构化答案，而不是干巴巴回一句
- 新模型的"基础体感"在升级——RAG、联网、工具调用、Markdown 排版，这些"看不见的细节"每次迭代都在变好

所以从知识获取角度，挑那个发布日期离今天最近的模型用就对了，理由比任何榜单都直接。复杂任务再回头看 benchmark，按需切换。

### 主流免费 Chatbot · 各自气质

| 产品 | 气质 | 免费力度 | 最强场景 |
|------|------|---------|---------|
| DeepSeek | 极客感 | ★★★★★ | 代码、数学、长链推理 |
| 通义千问 | 全能型 | ★★★★☆ | 多模态、办公、翻译 |
| 智谱清言 | 稳重学院派 | ★★★★☆ | 逻辑推理、Agent |
| Kimi | 书卷气 | ★★★★☆ | 长文档阅读、研报摘要 |
| 文心一言 | 知识检索派 | ★★★★☆ | 知识问答、行业资料 |
| 腾讯元宝 | 办公伙伴 | ★★★★☆ | 微信生态、文档协作 |
| 豆包 | 活泼亲民 | ★★★★☆ | 中文创作、短视频脚本 |

### 「免费时代」正在悄悄翻篇 · 2026.05 豆包率先收费

2026 年 5 月 4 日，豆包在 App Store 页面新增三档付费订阅——标准版 68 元/月、加强版 200 元/月、专业版 500 元/月，消息直接登上微博热搜，被业界解读为「国产大模型免费时代结束的标志性事件」。免费版仍保留，但旗舰能力与生产力场景全面付费化。所以本轮表格里，豆包从五星调为四星。

放眼整个国内梯队，目前真正能称为「完全免费 · 旗舰常驻」的，几乎只剩两家：

**DeepSeek**——网页版 chat.deepseek.com 一直免费，2026.04.24 上线的 V4 预览版（百万上下文、Pro 旗舰 + Flash 轻量）也是直接免费开放，没有付费墙、没有套餐分级。这是当前国产里最纯粹的「免费 + 旗舰」组合。

**通义千问**——也免费，但走的是「限时旗舰」路子：每当新旗舰发布（现在最新是 Qwen3.7-Max），免费 App 端会先放上让大家试用，体验一段时间、有了体感之后，就会把它换下去，降级成非旗舰。想长期稳定用最新版，还得走阿里云百炼或 API。跟 DeepSeek 那种"免费 + 旗舰长期在线"相比，千问是「能尝到最新，但留不住」。

未来 12 个月这条线会越缩越窄。免费 ≠ 永远免费，能用的时候要好好用。

### 那些「不在主页上」的好东西

很多模型厂商把最好的体验藏在了二级域名里——找到这些「隐藏入口」，体验立刻不同：

**Qwen Chat / Studio（阿里）**：访问 chat.qwen.ai，可自由切换 Qwen 全系列（含视觉、推理、Coder），界面比主品牌入口干净十倍，免费额度大方。

**Z.ai（智谱海外品牌）**：chat.z.ai 是 GLM 的国际版界面，无注册即用、支持 Agent 与浏览，对国内用户也友好。

> 工具的好坏，往往不在功能表上，而在每一次开屏的那 0.5 秒——它让你想用，还是不想用。

## 五、聚合：API 调用与「交叉验证」的智慧

单一模型有局限，多模型互证才接近真相。当你跨越了「Chatbot 用户」的边界，就来到了 API 与聚合客户端的世界——这里没有 IP 围墙，只有 token 与算力的交易。

### 为什么要走 API

- **突破限制**：网页版有上下文长度、对话轮数、并发上限；API 没有
- **自动化**：写脚本批处理文档、批量生成内容、构建 Agent，全靠 API
- **统一界面**：通过聚合客户端，在同一个窗口内调用十几个模型，告别频繁切换网页
- **成本可控**：按 token 计费，重度用户通常比开会员更便宜

### 多模型聚合客户端 · 一站调度

**Cherry Studio（桌面客户端）**：开源跨平台，集成 OpenAI、Claude、Gemini、DeepSeek、Qwen 等数百个模型。支持知识库、Agent、绘画，同一个问题可同时让 3 个模型作答横向对比，是国内最受欢迎的多模型工作台之一。

**硅基流动 SiliconFlow（API 聚合云）**：把 100+ 开源模型封装成统一的 OpenAI 兼容接口，siliconflow.cn。新用户免费额度足够上手，是 Cherry Studio 等客户端最常配的「弹药库」。

### 多模型交叉验证 · 一种思维方式

同一个事实问题，用 DeepSeek、Qwen、GPT 各问一遍，三家答案的交集是高置信区，分歧点恰恰是值得深挖的疑点。这种用法天然消解了单模型的「幻觉」与「立场」，特别适合做研究、做投资、做关键决策。

> 一个模型告诉你的是答案；多个模型告诉你的是「答案的方差」。方差，比答案更接近真相。

### 插曲 · Cherry Studio 的隐私争议（2026.04）

2026 年 4 月中旬，社区曝出 Cherry Studio 在用户已关闭「自动更新」「匿名错误报告」「开发者模式」的情况下，仍会上传应用启动、检查更新等设备信息（含设备指纹）。官方随后承认开关失效，归因于代码重构期的逻辑疏漏，已修复。聊天内容本身未被上传。

启示：开源不等于免审计。哪怕是社区好评的工具，对涉及隐私的开关也应每次更新后重新核验。对企业敏感数据，本地部署 + 离线模型仍是终极答案。

## 六、在地：本地部署与自主可控的底气

当数据不能出企业、当电力成为新石油、当美国制裁芯片，「本地部署」就从极客玩具变成了国家命题。

### 为什么要本地部署

- **数据安全**：医院、律所、研究院、金融、军工，原始数据不能离开内网
- **算力自主**：长期、高并发的推理任务，自建比租用更划算
- **定制能力**：本地微调、知识库私有化、Agent 编排自由
- **断网可用**：野外、车端、舰船、变电站等离线场景刚需

### 国家算力一张网

2026 年 3 月，「东数西算」之后的「算力六张网」正式成型，全年初步投入超 7 万亿元。同月，深圳点亮全国首个万卡级全栈自主可控智算集群——14000P 算力，全部基于华为昇腾 910C + CANN 软件栈。工信部首次提出「算力银行」「算力超市」概念，让中小厂商也能买得起、用得上算力。

这场底层重塑的意义在于：未来你部署的每一个本地模型，背后跑的可能就是国产芯片。DeepSeek V4 已经率先官宣「携手昇腾」，给整个生态打了样。

### 两个最常用的本地工具

**Ollama（命令行 / 极客向）**：一行 `ollama run deepseek-r1:7b` 就能跑起来，模型库丰富，对接生态最广（OpenWebUI、Cherry Studio、LangChain 全支持）。适合喜欢命令行、追求灵活配置的用户。

**LM Studio（图形界面 / 小白友好）**：纯 GUI 体验，模型搜索、下载、聊天、API 服务一站完成。Mac 上 MLX 加速尤其香，Windows/Linux 也有 CUDA/Vulkan 支持。第一次接触本地大模型，从这里开始。

关于教程，直接去 B 站搜「Ollama 部署」「LM Studio 入门」就行，上面有很多讲得清楚又不啰嗦的视频——从硬件选购到一键部署到知识库搭建，跟着做一遍就会了。

> 本地部署的真正价值，不是省钱，是让你拥有「可以不依赖任何人」的那种笃定。

## 七、权衡：哥德尔不完备定理与一切选型的终极智慧

1931 年，25 岁的库尔特·哥德尔用一篇论文摧毁了希尔伯特的数学大一统梦想。他证明了：任何足够强大的形式系统，都必然存在「真但不可证」的命题，且无法在系统内证明自身的一致性。这不是数学的失败，而是认知诚实的开端。

### 不完备定理与大模型选型

把大模型看作一个「形式系统」，哥德尔的洞见会令人脊背一凉：

- 没有任何一个模型能同时做到「最强、最快、最便宜、最隐私、最易部署」
- 追求多模态完备，必然牺牲专精；追求开源开放，必然受限于训练资源；追求绝对隐私，必然让渡云端能力
- 每一次「全能模型」的承诺，本质上都是营销，不是科学

### 不可能三角 · 大模型版

**能力强**（参数大、上下文长、多模态、推理深）→ 必然依赖云端 + 高成本

**成本低**（免费使用 / 低单价 token / 低硬件要求）→ 必然牺牲性能或隐私

**隐私好**（本地部署 / 数据不出网 / 可审计）→ 必然受限于本地算力

三者之中，挑两个，永远是工程的真相。

### 所以，真正的选型方法论

1. **从需求倒推**，不要从模型正推。先问：是写作？是代码？是知识库？是 Agent？是离线场景？
2. **分级配置**：日常 80% 用免费 Chatbot，重要 15% 用 API + 多模型交叉验证，敏感 5% 走本地部署。
3. **不押宝单模型**：模型迭代极快，今天的王者三个月后可能让位。永远保留切换能力。
4. **验证而非信任**：关键事实必交叉验证；关键代码必本地审；关键工具必读源码或开关。
5. **拥抱国产生态**：在算力自主可控成为国家底盘的今天，理解并使用国产模型，本身就是一种长期主义。

> 哥德尔告诉我们：完美的系统不存在。工程师告诉我们：可用的系统就够了。这两句话，是大模型时代最该挂在墙上的座右铭。

「真正的智慧，不是找到那个最强的模型，而是知道在何时、为何、以何种代价，使用哪一个。」
